## Getting Started

### Login Credentials
- **Username:** `admin` or `mc`

### Quick Start
1. Open `standalone.html` in a web browser
2. Login with admin credentials
3. Data is saved automatically to your browser's localStorage

### CSV Upload Format
Participants CSV should have columns:
- Name, Designation, Organization, Phone, Email, Sports
